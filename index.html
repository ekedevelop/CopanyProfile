BELAJAR GIT

CREATE

ls -al = Untuk apa belum tau

git init = Untuk menginisialisasi file git

git clone = Untuk mengcloning file git yang open source, atau untuk menerjemahkan file yang sudah diupload ke github oleh rekan kerja ke program

MAKE A CHANGE

git add "nama_file"= Untuk menambahkan file program yang baru dibuat ke dalam file Git sebelumnya

git add . = Untuk menambahkan beberapa file program dan/atau folder yang baru dibuat ke dalam file Git sebelumnya

git commit -m "keterangan apa yang diubah"= Untuk memastikan semua file sudah masuk kedalam history

git reset "nama_file" = Untuk mengembalikan ke file sebelumnya atau menghapus perubahan. setelah direset maka file yang direset tidak akan terbawa oleh file git

git reset --hard = Untuk mengembalikan semua file ke file sebelumnya atau menghapus perubahan. setelah direset maka file yang direset tidak akan terbawa oleh file git

OBSERVE

git status = Untuk melihat status perubahan file Git

git diff = Untuk mengetahui apa yang berubah sebelum di commit

git show = Untuk menampilkan history perubahan

git log = Untuk memastikan git commit berhasil atau tidak

git remote add origin + "link github" = Untuk repositori Git yang dihosting di Internet atau jaringan lain.

git remote -v = Untuk cek apakah file offline dan online sudah terintegrasi atau belum.

SYNCRO, untuk sinkronisasi antara lokal dan server

git push = Untuk mengupload dari lokal ke online

git pull = Untuk menarik perubahan dari online ke lokal

git fetch = Untuk mengunduh konten dari repositori remote, mengambil resource dari online tanpa mengganggu file lokal/laptop/pc
